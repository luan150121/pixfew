# Arquivos legados

Esta pasta guarda arquivos da engine antiga do PIXFEW.

Eles foram mantidos apenas como referencia historica para consulta, comparacao
ou recuperacao de alguma ideia antiga.

A engine oficial atual usa `game.html` junto com os modulos da pasta `js/`.

Novos sistemas, salas, objetivos, transicoes ou correcoes nao devem ser
implementados nestes arquivos antigos.
